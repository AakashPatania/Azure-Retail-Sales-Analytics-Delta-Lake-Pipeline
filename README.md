**PROJECT OVERVIEW**

This project demonstrates a fully functional end-to-end Azure data engineering pipeline using the Adventure Works dataset. The solution follows a modern Lakehouse architecture and includes data ingestion, raw and transformed data storage, ETL processing, data warehousing, and business intelligence reporting.

The goal of this project is to convert raw enterprise data into meaningful business insights related to sales performance, customers, product categories, territories, and calendar trends.


**DATA MODEL**
 
The dataset includes the following entities:

Customers

Products

Product Categories

Product Subcategories

Sales Territories

Calendar / Date dimension

**KEY FEATURES**

Real-time data ingestion using Azure Data Factory HTTP connector

Distributed data processing and Delta Lake implementation in Azure Databricks

Lakehouse Bronze, Silver, Gold architecture for scalable refinement

Data warehousing and analytical modeling using Synapse SQL

Power BI dashboards for business insights and decision support

**TECHNOLOGIES USED**

Azure Data Factory

Azure Data Lake Gen2

Azure Databricks (PySpark & Delta Lake)

Azure Synapse Analytics

Power BI

Adventure Works Dataset (HTTP API Source)

**FINAL OUTPUT**

Interactive Power BI dashboards showcasing:

Sales trend analysis

Top customers and regions

Product performance by category and subcategory

Time-based seasonality insights
